## 技术架构

```
后端：Python + FastAPI
  ├── DeepSeek API（AI大模型，model: deepseek-chat）
  ├── Tavily Search API（实时搜索）
  └── 托管前端静态文件（backend/static/index.html）

前端：纯 HTML / CSS / JavaScript（无框架）
  └── BASE_URL = ''（同域请求，不写死地址）

部署：Railway
  ├── Root Directory: backend
  ├── Start Command: uvicorn main:app --host 0.0.0.0 --port $PORT
  └── 自动 CI/CD（推送 GitHub 自动部署）
```

## 简历项目描述（中文版）

**职前探 · AI 求职备考助手**｜个人独立项目｜2025.03 至今

**项目背景**
针对求职者"不了解目标公司动态、看不懂 JD 要求、不知道面试考什么"的核心痛点，独立设计并开发一款一站式求职备考工具。

**我的角色**
产品负责人 + 独立开发者（全程使用 Vibe Coding 方法，零编程基础，通过与 AI 对话驱动完成完整产品研发链路）

**核心功能**
- 🔍 实时搜索目标公司近期动态与新闻
- 📋 智能解析招聘 JD，提炼岗位核心能力要求
- 💬 根据公司和岗位自动生成高频面试题及备考建议
- 📄 针对目标岗位提供个性化简历优化建议

**技术实现**
- 后端：Python + FastAPI，接入 DeepSeek API（大语言模型）+ Tavily Search API（实时搜索）
- 前端：HTML / CSS / JavaScript，响应式设计，支持手机与电脑访问
- 部署：Railway 云平台，GitHub 自动 CI/CD，公网可访问

**项目成果**
- Web 端已正式上线，任意用户无需注册即可使用
- 微信小程序版本已开发完成，正在审核中
- 项目已发布至 Product Hunt，代码开源于 GitHub

**项目地址**
